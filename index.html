Final Year Project:

Hossain's final year project was about the day to day issues of trying to use technology to benefit you without the need to download multiple applications. His goal was to build a cross-platform application to a target audience who go to college or university. The application focused on a few major areas that Students struggle with. The sections of the application were Tasks, Projects, Exercise, Money Management and Notes. The application provided all these features and functionality without the need to download multiple apps. All in one mobile application and built with one code base for the two largest mobile OS which is Apple and Android. The project consisted of both a back end which was developed with Firebase and the front end which was developed with Flutter and Dart.
